# Product CRUD Web Application

A practice-based simple web application built using **Spring MVC and Hibernate** to perform CRUD operations on products.

## Tech Stack
- Java
- Spring MVC
- Hibernate
- Maven
- MySQL
- Apache Tomcat
- JSP

## Features
- Add products
- View product list
- Update products
- Delete products

## Purpose
This project was created to learn Spring MVC architecture and Hibernate ORM integration.
